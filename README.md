## Startup instruction

To start app, run:

```
npm install
npm start
```

Runs the app in the development mode.<br>
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

To start tests, run:

```
npm test
```

App is available at:
[https://abm0.github.io/map-routes](https://abm0.github.io/map-routes)
